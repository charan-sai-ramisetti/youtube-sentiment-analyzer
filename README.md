# YouTube Comment Sentiment Analyzer

A web application that analyzes YouTube video comments and classifies them as **Positive**, **Negative**, or **Neutral** using NLP.

---

## Features
- Accepts any YouTube video link  
- Fetches up to 1000 comments  
- Performs sentiment analysis  
- Displays results in a pie chart  
- Simple frontend and backend  

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Python, Flask  
- NLP: VADER Sentiment Analyzer  
- API: YouTube Data API v3
